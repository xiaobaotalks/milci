mi-cc

终端里的智能编程助手 · 工具调用 · 四层记忆 · 分层摘要压缩 · MCP Server

⭐ Star on GitHub 🚀 快速开始

核心特性

为发烧而生的极简 AI 编程工作流

🧠

四层记忆系统

checkpoint（会话） / MEMORY（项目） / notes（草稿） / history（日志），启动自动恢复，跨会话持久化。

📦

分层摘要压缩

三档触发（60% / 80% / 95%），保留最近 N 轮原文，更早内容升层为摘要，自动持久化到 compress-state.json。

🤖

13 个模型自动识别

内置 MiMo 1M / GPT-4o / Claude / DeepSeek / GLM-4 / Moonshot 上下文窗口映射，启动自动设置 maxTokens。

🛠️

工具调用 + MCP

内置 readFile / writeFile / runShell / git，支持 MCP 风格外部工具 JSON 声明，shell 注入自动转义。

📚

技能库自动蒸馏

/distill 从历史挖掘高频工作流，top-2 匹配技能自动注入 System Prompt。

⌨️

Tab 补全 + 危险命令拦截

斜杠命令一级 + 二级补全，8 条 regex 拦截 rm -rf / · curl|bash · fork bomb 等高危命令。

快速开始

3 步即可在任何终端运行

1克隆并安装依赖

git clone https://github.com/xiaobaotalks/mi-cc.git
cd mi-cc
npm install

2配置 API（默认对接小米 MiMo）

cp .env.example .env
# 编辑 .env，填入 API_KEY，MODEL=mimo-v2.5-pro

3启动

npm run dev
# 或全局安装
npm link && mi-cc

斜杠命令

输入 / 然后按 Tab 即可补全

命令说明
/connect设置 API 配置并写入 .env
/compact手动压缩上下文
/distill经验蒸馏，生成技能库
/dream记忆整理 + 清理过期日志
/skill查看/刷新技能库（list / name / reload）
/tools查看所有可用工具
/help显示帮助
/exit退出程序
